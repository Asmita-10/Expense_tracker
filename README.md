# 🧾 Expense Tracker

A pastel-themed personal expense tracker built with vanilla HTML, CSS & JavaScript. Log daily expenses by name, amount, day, and month — with a live balance summary, income overview, and scrollable expense history. Clean sidebar profile, soft UI, and fully responsive layout.

---

## ✨ Features

- **Live Summary Cards** — Balance, Income, and Expenses update instantly as you add entries
- **Expense Form** — Add an expense with a name, amount, day of the week, and month
- **Expense History** — Newest entries appear at the top with name, date metadata, and amount
- **Profile Sidebar** — Displays username, total spent this month, and entry count
- **Pastel UI** — Soft peach, sage, and lavender palette with warm cream background
- **Responsive Design** — Works on desktop and mobile screens

## 🚀 Getting Started

No build tools or dependencies needed. Just open locally:

1. **Clone the repo**
   ```bash
   git clone https://github.com/your-username/expense-tracker.git
   cd expense-tracker
   ```

2. **Open in browser**
   ```bash
   open exp.html
   ```
   Or simply double-click `exp.html` in your file explorer.

---

## 🖥️ Usage

1. Enter an **expense name** (e.g. Groceries, Transport)
2. Enter the **amount** in ₹
3. Select the **day** and **month**
4. Click **+ Add Expense**

The summary cards and sidebar stats update automatically. Entries are listed newest-first.

## 🎨 Design

The UI draws from a warm budget-planner aesthetic — pastel cards, serif headings, and soft shadows. Color roles:

-  **Peach/Blush** — Expenses & primary actions
-  **Sage** — Balance
-  **Lavender** — Income
-  **Cream** — Page background
